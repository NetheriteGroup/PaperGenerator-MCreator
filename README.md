# Paper Generator for MCreator 2026.1
An unofficial plugin for [MCreator](https://mcreator.net/), a code generator for Minecraft. This plugin adds a Paper generator.
**The current targeted Minecraft version is 1.21.6** (Based on the original framework by Goldorion).
## Features
* Commands
* Procedures
* Custom elements
* Custom enchantments
## Information
- Make sure you use MCreator 2026.1
- Plugins also require [Spigot](https://getbukkit.org/download/spigot) to function properly.
